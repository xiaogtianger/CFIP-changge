# Clean IP List 优质 IP 优选库

**自动每日更新** 的高质量 IP 列表（支持 JP、KR、SG、US 地区）。

通过 GitHub Actions 定时从多个来源采集 IP，经过**去重、智能测速优选**后，生成干净可用的 IP 列表。

---

## ✨ 项目特点

- 每日自动更新（北京时间早上 5:00）
- 支持手动触发更新
- 智能优选逻辑：
  - 有测速数据时 → 每个地区取 **最快前 10 个**
  - 无测速数据时 → 每个地区随机取 **8 个**（或全部）
- 最终输出格式：`IP:端口#地区`
- 已排除常见问题网段（可根据需要调整）
